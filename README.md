## 📖 О проекте

Этот проект представляет собой учебный стенд для автоматизированного тестирования интернет-магазина [demo.litecart.net](https://demo.litecart.net/). Он был разработан в рамках дисциплины по тестированию программного обеспечения. Главная цель — показать на практике, как можно настроить инфраструктуру автотестов с использованием Selenium WebDriver и JUnit 5, а также продемонстрировать различные подходы к тестированию веб-приложений в разных браузерах.

> **Примечание**: тесты используют демо-версию магазина LiteCart, которая не требует дополнительной настройки или развертывания.

## 🚀 Возможности и особенности

*   **Кросс-браузерное тестирование**: Проверка работоспособности сайта в браузерах **Firefox** и **Chrome**.
*   **Поддержка JUnit 5**: Использование современных подходов к написанию тестов (параметризация, расширения, управление порядком выполнения).
*   **Умные ожидания**: Применение `WebDriverWait` для повышения стабильности тестов.
*   **JavaScript-расширения**: Визуальная подсветка элементов на странице и выполнение кастомных скриптов.
*   **Встроенные отчёты**: Автоматическое логирование хода выполнения тестов с помощью JUnit-расширений.
*   **Работа с DOM**: Поиск элементов по XPath, CSS-селекторам и другим стратегиям.

## 📥 Установка и запуск

### Требования

Убедитесь, что на вашем компьютере установлены:
*   **JDK 21** или новее.
*   **Apache Maven** (версия 3.6+).
*   Браузеры **Mozilla Firefox** и/или **Google Chrome**.

### Инструкция

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Sovynhik/RSREU-test-litecart-with-selenium.git
    ```
2. Перейдите в директорию проекта:
    ```bash
    cd RSREU-test-litecart-with-selenium
    ```
3. Выполните команду для запуска всех тестов:
    ```bash
    mvn clean test
    ```
4. Для запуска конкретного тестового класса:
    ```bash
    mvn test -Dtest=LitecartBrowserCompatibilityTest
    ```
## 🧪 Сценарии тестов

| Каталог | Назначение |
| :--- | :--- |
| `src/test/java/lab_01` | **Юнит-тестирование**. Проверка калькулятора академической стипендии. Включает параметризованные тесты, проверку исключений и работу с `@Timeout` и `@Disabled`. |
| `src/test/java/lab_02` | **Кросс-браузерный тест**. Проверка перехода на карточку товара "Purple Duck" в Firefox и Chrome. |
| `src/test/java/lab_03` | **XPath и визуализация**. Поиск ключевых элементов (логотип, корзина, форма поиска) с помощью XPath, подсветка найденных элементов и сохранение скриншотов. |
| `src/test/java/lab_05` | **E2E-тест жизненного цикла**. Полная проверка сценария: регистрация нового пользователя, его авторизация, редактирование профиля и последующее удаление через админ-панель. |
